HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It is very easy to make mistakes. Go slow and be thorough.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is used to make the stylized changes to the website aesthetic.
  • Style settings can be set under a ".card" to be applied in specific places on the webpage.

Git

  • Be careful not to use git commands without being certain of their purpose.
  • git status: checks what branch is currently being worked on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes made to the staging area of git
  • git commit -m "message": adds a message associated with the committed changes.
  • Be sure to pull the main branch before pushing changes.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript largely executes behind the scenes of the website, but also dictates what happens when a user interacts with the page.
